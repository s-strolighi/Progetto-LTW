About the Website

Quando abbiamo cominciato a lavorare al nostro progetto eravamo in pieno lockdown, da questa situazione è nata la nostra idea di racchiudere in un sito web informazioni e consigli su come poter impiegare il tempo stando a casa.
Parlando dell'implementazione, per la parte riguardante lo stile abbiamo usato componenti della libreria css di Bootstrap, oltre a fogli di stile css da noi sviluppati.
Per l'interazione con la pagina abbiamo utilizzato file javascript, contenentii anche codice JQuery, da noi implementati e personalizzati.
Abbiamo inoltre utilizzato Ajax per effettuare chiamate al server esterno per implementare alcune funzionalita del sito.
Infine sono presenti due file php usati per l'interazione con il database sql, anch'esso da noi implementato usando PostgreSQL, allo scopo di integrare nel sito servizi di registrazione e log-in.
